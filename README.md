# Digital Effect 🚀

**Digital Effect** is a modern digital marketing agency website built with Next.js — designed to attract clients, showcase services, and convert visitors into leads.

---

## 🌐 Overview

Digital Effect is the online presence for a full-service digital marketing agency. The site communicates brand authority, presents core services, and drives client engagement through a clean, high-performance web experience.

---

## ✨ Features

- **Services showcase** — SEO, social media, paid ads, content marketing, and more
- **Portfolio / case studies** — Real results for real clients
- **Lead generation** — Contact forms and CTAs optimized for conversion
- **Responsive design** — Seamless experience on all devices
- **Fast performance** — Built on Next.js with SSR/SSG for optimal load times
- **SEO-ready** — Meta tags, Open Graph, and structured data out of the box

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| [Next.js](https://nextjs.org/) | React framework (SSR / SSG) |
| React | UI component library |
| CSS Modules / Tailwind | Styling |

## 🚀 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-org/digital-effect.git

# Navigate into the project
cd digital-effect

# Install dependencies
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm run start
```

---

## 🌍 Deployment

The site is optimized for deployment on **Vercel**:

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Alternatively, any Node.js-compatible hosting platform works.
